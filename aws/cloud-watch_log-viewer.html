Export logs with:

Export logs with filter:

        aws logs filter-log-events \
  --log-group-name XXX-prod-log-group-ecs-be \
  --region eu-central-1 \
  --start-time $(date -d '24 hours ago' +%s000) \
  --output json > /tmp/ecs-logs-24h.json
